This guide assumes you have already completed the following steps:

  • Connected SkyConnect.
  • Have the Zigbee Home Automation (ZHA) integration up and running.
  • Multiprotocol is not available on current software versions. These steps can only be done on Home Assistant 2024.4 and older versions.
---

Multiprotocol is an experimental feature and is not recommended for use. For more information, read about the issues with multiprotocol.
