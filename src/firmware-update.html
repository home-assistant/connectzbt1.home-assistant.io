Firmware update

First things first: a firmware upgrade is not required for SkyConnect at this point.

We're also working on making firmware updates available via Home Assistant. So you will receive updates automatically in the future. These updates are still under development.

SL Web Tools to install firmware updates

If you're feeling adventurous and want to try out the latest (experimental) firmware update, you can use the web-flasher below to install the firmware directly from your web-browser over USB.
