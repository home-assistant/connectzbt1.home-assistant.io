Web-Flasher Firmware updates for Connect ZBT-1

Firmware update

A firmware upgrade is not required for the Home Assistant Connect ZBT-1 at this point. There are no known bugs with the firmware it ships with.

We're working on making firmware updates available via Home Assistant. You will receive updates automatically in the future.

SL Web Tools to flash firmware updates from the browser

This page is for troubleshooting. If you use Connect ZBT-1 in Home Assistant, the Home Assistant discovery flow will automatically install the correct firmware. You do not need to install firmware using this tool. Furthermore, on every start of the OpenThread Border router, the latest Thread firmware which is bundled with the add-on will be installed.

If for some reason you like to flash a firmware still: you can use the web-flasher below to install the latest firmware version for Home Assistant Connect ZBT-1 directly from your browser over USB.

The firmware is the same for both Home Assistant SkyConnect and Connect ZBT-1. You can still use the button below for both models.

To read more about the SL Web Tools, refer to this blog post.
