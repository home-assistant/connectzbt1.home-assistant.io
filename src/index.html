Zigbee and Thread USB stick by the creators of Home Assistant

Home Assistant SkyConnect is the easiest way to add Zigbee to your Home Assistant instance. SkyConnect is also an open-source-friendly path to add a Thread border router. Some Matter devices require a Thread network. The Home Assistant developers provide regular firmware updates for SkyConnect to make sure the device stays up-to-date and ensure compatiblity with Home Assistant.

Zigbee

The SkyConnect comes with a Zigbee firmware pre-installed. Home Assistant officially supports ZHA (Zigbee Home Automation), the Zigbee stack integrated into Home Assistant Core. The following guides describe how to use SkyConnect with ZHA.

Thread

Home Assistant SkyConnect can also be used to turn your Home Assistant installation into a Thread border router. Currently, enabling Thread involves manual steps. The integration of the Home Assistant Thread border router with Matter is work-in-progress. To learn more about Thread border routers, refer to the List of Thread border routers. To learn how to use SkyConnect to turn your Home Assistant into a Thread Border router, refer to Turning Home Assistant into a Thread border router.
